DESCRIPCION DE LOS FICHEROS
---------------------------
	
- ContratacionColectivos.wsdl
Descriptor del servicio Web que incluye diversas funciones de apoyo al registro y firma de colectivos
diseñadas para ser utilizadas desde los aplicativos de contratación de las Entidades. 

- ContratacionColectivos_con_cabeceras_seguridad.wsdl
Idem ContratacionColectivos.wsdl pero con cabecera de seguridad WS-Security, para
manejar la seguridad a través de los bindings de XML generados en cada lenguaje
de programación.

- policy.xml
Política de seguridad según especificación WS-SecurityPolicy con UsernameToken
de Oasis.

- AcuseRecibo.xsd
Esquema que define la estructura del XML que se devuelve ante una petición de validación, registro,
consulta de colectivos, o un envío de documentos de colectivo. 

- ListaColectivos.xsd
Esquema que define la estructura del XML que se devuelve ante una petición de consulta de Lista de Colectivos.

- Tipos.xsd
Esquema que contiene el diccionario de tipos común de Agroseguro.

- xmime.xsd
Esquema de validación de tipos XMLMIME de W3C
